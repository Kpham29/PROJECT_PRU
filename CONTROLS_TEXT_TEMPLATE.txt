  GAME CONTROLS - Copy vào Unity Text
===============================================

--- VERSION 1: SIMPLE TEXT ---

MOVEMENT
A / ← : Move Left
D / → : Move Right
Space : Jump
Shift : Run/Sprint

COMBAT
Mouse Click : Basic Attack
Q : Special Skill
E : Shield/Block

SYSTEM
ESC : Pause Menu
Tab : Map

OBJECTIVE
🎯 Defeat all enemies and reach the portal!
⭐ Collect coins for bonus points
💀 Watch out for traps and boss attacks

  CÁCH SỬ DỤNG
===============================================

1. Copy text version bạn thích
2. Trong Unity:
   - Tạo Text hoặc TextMeshPro
   - Paste vào
3. Adjust font size, color
4. Done!
